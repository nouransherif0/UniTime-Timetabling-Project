## Step 2: Textual Description of What UniTime Does
UniTime is an open-source web-based scheduling system built for universities and colleges.
It helps academic institutions build and manage course timetables and exam schedules in a conflict-free way.

The system handles four main areas:
- Course Timetabling: assigns courses to time slots and rooms based on constraints and preferences
- Exam Timetabling: schedules final and midterm exams without student conflicts
- Student Scheduling: assigns students to class sections while minimizing schedule conflicts
- Event Management: supports booking rooms for university events beyond regular classes

The system supports multiple roles including system administrators, departmental schedule managers,
and students, each with different levels of access and control.
