# Phase 1 - Point 4: Feature Descriptions

## Feature 1: Email Notification for Timetable Changes
**Type:** New Feature
**Component:** Course Timetabling & Student Scheduling
When a timetable change occurs, the system will automatically 
send an email to all affected students and instructors.

## Feature 2: Export Timetable to PDF
**Type:** New Feature  
**Component:** Course Timetabling & Student Scheduling
Students can export their timetable as a downloadable PDF file.

## Feature 3: Search/Filter Classes by Course Name
**Type:** Enhancement
**Component:** Course Timetabling
A search bar to filter displayed classes by course name.

## Feature 4: Add Notes/Comments to a Class
**Type:** New Feature
**Component:** Course Timetabling & Event Management
Instructors can attach short notes to any scheduled class.

## Bug 5: Duplicate Class Detection
**Type:** Bug Fix
**Component:** Course Timetabling
Detects when the same course is scheduled in the same room 
at the same time slot.
